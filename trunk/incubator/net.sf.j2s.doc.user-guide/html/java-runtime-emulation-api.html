Java Runtime Emulation API

In the section called “Doing java in HTML documents” we have examined how a java program is executed in an html document using the object ClazzLoader provided by J2S that contains method for configuring the java runtime with javascript, like setting the java classpath, loading classes and invoking java code. Here we will document all relevant functions of that object.
