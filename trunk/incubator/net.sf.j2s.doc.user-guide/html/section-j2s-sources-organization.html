Source organization

As said in previous sections, Java2Script plugin contains both .java files that will be compiled into .class like any other Java application like projects , but also contains a lot of .java files, this will be compiled by Java2Script compiler itself to JavaScript files.

plugin related projects.  net.sf.j2s.core, net.sf.j2s.lib, net.sf.j2s.ajax net.sf.j2s.ui. TODO: explain more

javascript related projects.  net.sf.j2s.java.core, net.sf.j2s.java.junit, net.sf.j2s.org.eclipse.swt. TODO: explain more

Both kind of projects will be built automatically [4] so we only have to save our files to impact the changes. Nevertheless, if we are modifying files that will be compiled to JavaScript, like project net.sf.j2s.java.core, we should also rebuild j2slib. It is easy. Just open net.sf.j2s.lib/build/build.xml in Ant view and run the j2s.pack.lib ant task:
