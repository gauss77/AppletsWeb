The following is a list with summaries about each chapter contents:

  • Chapter 2, Introduction Introduces Java2Script product

  • Chapter 3, Getting Started Teach the user, step by step how to create its first Java2Script application, how to execute it in the browser, and the basics about Java2Script application projects.

  • Chapter 4, Code Translation  deepens on how java code is translated by Java2Script compiler into JavaScript. Details how a Java2Script application is loaded in an html document with javascript.

  • In Chapter 5, The J2S plugin  we give detailed description of all functionalities of Java2Script eclipse plugin, reviewing each component of the plugin and each plugin capability.

  • In Chapter 6, Java Support  we will talk about java language and standard library support in javas2cript. An detailed list of each supported class and method of java.lang, java.util, java.io, etc is presented.

  • Chapter 7, Integrated Java libraries  explains how to use java libraries supported by Java2Script like SWT, jUnit, etc.

  • Chapter 8, Customizing output code with @j2s JavaDoc tags and annotations  explains how to customize Java2Script compiler output using JavaDoc @j2s special tags.

  • Chapter 9, Working With native code gives details about using JavaDoc tag @j2sNative for embedding native javascript code. Gives information and recommendations about using native objects in java code.

  • Chapter 10, Client - Server communication Describe Java2Script facilities for client-server communication, like RPC, simple pipe, comment and doing normal ajax in Java2Script applications.-

  • Chapter 11, Getting started with Java2Script sources contains step by step guide on working with Java2Script plugin sources so the user may debug the plugin and customize it or simply know the plugin's internals.

  • Appendix A, J2S Java Emulation API gives detailed information about Java2Script java language and java runtime javascript emulation, documenting methods of Clazz for java language emulation and ClazzLoader for java runtime emulation.

  • Appendix B, GNU Free Documentation License contains the full text of the license used for releasing this book.
